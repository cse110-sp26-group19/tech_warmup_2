# AI Plan Tech Warmup II

## 1. AI tool
Claude Code

## 2. Why this one
We used Claude in the last warm up and the team is already comfortable with it.

## 3. Model Selection
Claude sonnet 4.6 good balance between speed and code quality.

## 4. Prompting Strategy
- One small feature per prompt
- Ask the AI to explain its reasoning before writing code
- Include examples in prompts when possible

## 5. Error Handling
- Try 3 different prompts before giving up
- Log every failure in the AI log
- Hand-edit only as a last resort

## 6. Hand Editing Policy
Only hand-edit after 3 failed prompt attempts. Log the reason.
